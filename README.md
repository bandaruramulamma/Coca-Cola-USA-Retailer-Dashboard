# Coca-Cola-USA-Retailer-Dashboard
The Coca-Cola USA Retailer Dashboard is an interactive Power BI report designed to analyze sales performance across different retailers, regions, and product categories in the United States. The dashboard provides clear insights into key business metrics such as total sales, profit, units sold, and operating margins.
# Coca-Cola USA Retailer Dashboard 📊

## 📌 Project Overview
This Power BI dashboard analyzes Coca-Cola sales performance across retailers, regions, and products in the USA.

## 🎯 Objectives
- Analyze sales and profit trends
- Identify top-performing retailers
- Understand regional performance
- Track product-wise sales

## 📊 Key Metrics
- Total Sales
- Total Profit
- Units Sold
- Operating Margin

## 📍 Dashboard Features
- Regional Analysis
- Retailer Comparison
- Time-based Trends
- Interactive Filters
